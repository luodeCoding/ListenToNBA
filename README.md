# iOS ListenToNBA APP

* ListenToNBA有观看直播并支持后台播放收听NBA功能

#结构
*******
###本项目使用Object-C开发，共分为三个部分。

1. 基础页面布局（包含首页的Banner和列表展示以及webview相关内容）；
2. 数据方面（采用第三方SDK：LeanCloud，在平台端建表录入数据，APP端获取即可）；
3. 直播控件开发（围绕AVPlayer展开）
